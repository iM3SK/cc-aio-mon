# Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) v2.1.

## Summary

- Be respectful and constructive.
- Focus on the work, not the person.
- Issues and pull requests are the right place for technical discussion.

## Enforcement

Violations can be reported by opening a GitHub issue or contacting the maintainer directly. Reports will be reviewed and addressed promptly.
